## TC-FUNC-01
**ID:** TC-FUNC-01  
**Название:** Базовый поиск по валидному запросу  

**Предусловия (Preconditions):**  
API OpenLibrary доступен

**Шаги (Steps):**  
1. Отправить GET-запрос `/search.json?q=harry+potter`

**Ожидаемый результат (Expected Result):**  
- HTTP 200 OK  
- Ответ в формате JSON  
- JSON содержит ключи `numFound`, `docs`  
- `docs` не пустой

**Фактический результат:** 200 OK — PASSED

**Статус:** New

**Приоритет:** High
